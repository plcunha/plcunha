## About me

I turn operational problems into small, testable systems. My recent work spans resilient LLM routing, agent-loop reliability, backend APIs, CI/CD, and developer tooling.

- Building backend services and automation with a reliability-first mindset.
- Contributing fixes and operational documentation to [NousResearch/hermes-agent](https://github.com/NousResearch/hermes-agent/pulls?q=is%3Apr+author%3Aplcunha).
- Open to backend, API, developer tooling, and AI automation opportunities.

## Selected work

| Project | What it demonstrates | Stack |
| --- | --- | --- |
| [**hermes-free-ride**](https://github.com/plcunha/hermes-free-ride) | Discovers, probes, ranks, and rotates free OpenRouter and NVIDIA NIM models for Hermes subagents without replacing the primary model. | Python, APIs, resilient routing |
| [**Next.go**](https://github.com/plcunha/next.go) | A Go implementation of Next.js concepts with file-system routing, SSR, API routes, hot reload, and standalone binary builds. | Go, Gin, SSR, CLI tooling |
| [**Blog Auth API**](https://github.com/plcunha/blog-auth) | REST API with access/refresh JWTs, role-based access control, OpenAPI docs, migrations, containers, and CI. | NestJS, TypeScript, MySQL, Docker |
| [**RFID-GAME**](https://github.com/plcunha/RFID-GAME) | Maps RFID tags to configurable keyboard commands, with LLRP reader control, signal statistics, and a game mode. | Python, LLRP, hardware integration |
